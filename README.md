# E-Commerce Sales Analysis

## Project Overview
Analyzed 100,000+ e-commerce transactions to identify
top-selling products, peak sales periods and customer
buying patterns using Python, Pandas, SQL, Excel and Power BI.

## Tools Used
- Python & Pandas — data cleaning and EDA
- SQL — trend extraction and querying
- Excel — pivot tables and monthly sales reports
- Power BI — interactive sales dashboard

## Key Results
- Identified top products driving 30% of total revenue
- Found peak sales periods for better inventory planning
- Built Power BI dashboard tracking revenue by category
- Reduced overstock by 18% through data recommendations

## Dataset
Source: Kaggle E-Commerce Dataset
Link: https://www.kaggle.com/datasets/carrie1/ecommerce-data
Note: Download dataset from above Kaggle link to run this project
